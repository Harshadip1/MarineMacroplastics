# MarineMacroplastics
## Autonomous Sub Pixel Detection and Trajectory Mapping of Floating Marine Macroplastics
This project proposes a smart and automated system for monitoring, detecting, and managing marine plastic pollution using a combination of satellite imagery, artificial intelligence, drone technology, and mobile-based workforce coordination.

The system begins by acquiring ocean images from satellites such as Sentinel-2. These images are processed using machine learning algorithms capable of detecting plastic waste in ocean regions despite challenges like spectral confusion and sub-pixel detection. The system classifies affected areas into low, medium, and high plastic concentration zones.

Upon detecting plastic, the system automatically generates alerts and deploys drones for on-site verification. These drones capture high-resolution images and gather precise information about the quantity and location of plastic waste. The collected data is transmitted to a cloud-based server and displayed on a web dashboard accessible to government authorities and administrators.

Simultaneously, a mobile application notifies field workers about the affected areas, including details such as location, plastic density, and priority level. Workers are then dispatched to collect the plastic and transport it to designated distribution centers. These centers log the collected quantity, process the waste, and forward it for recycling or reuse.

The system ensures a continuous feedback loop, where all data—from detection to recycling—is updated in real time on the central platform. This enables efficient monitoring, decision-making, and large-scale management of marine plastic pollution.
